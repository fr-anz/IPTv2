# Personal Finance Analytics System

Python web application scaffold for the INTE 202 final project requirement: **Real-Life Data Analytics System using Python**.

The app analyzes the raw project dataset, cleans the data in memory, computes descriptive statistics, generates charts, and presents insights in a Flask dashboard. The original dataset is preserved as `Personal_Finance_Dataset.csv`.

## Features

- Loads the raw project CSV dataset or an uploaded CSV with the same core columns.
- Cleans and preprocesses records by renaming columns, converting data types, removing duplicates, filtering invalid rows, and normalizing text.
- Standardizes the added `Payment Method` attribute, including blank and inconsistent values.
- Adds derived analytics attributes such as period, quarter, signed amount, amount band, and a non-mutating outlier flag calculated separately for income and expense records.
- Computes totals, averages, median, mode, standard deviation, min, max, savings rate, category rankings, and monthly trends.
- Generates the required chart types: bar graph, line graph, pie chart, and histogram, plus a payment method expense chart.
- Exports a cleaned CSV for project submission.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
python -m pip install -r requirements.txt
```

## Run the Web App

```bash
python app.py
```

Then open:

```text
http://127.0.0.1:5000
```

## Create the Raw Project Dataset

Create the raw project dataset with the added `Payment Method` column and realistic data quality issues:

```bash
python scripts\create_raw_dataset.py
```

The script writes:

```text
data/raw/raw_personal_finance_dataset.csv
```

This script only creates the raw project dataset. It does not clean or process the data.

## Clean the Dataset

Only run this when you intentionally need the cleaned CSV output for submission:

```bash
python scripts\clean_dataset.py
```

By default, this reads:

```text
data/raw/raw_personal_finance_dataset.csv
```

and writes:

```text
data/processed/cleaned_personal_finance_dataset.csv
```

You can also choose paths explicitly:

```bash
python scripts\clean_dataset.py --input data\raw\raw_personal_finance_dataset.csv --output data\processed\cleaned_personal_finance_dataset.csv
```

## Dataset Attribute Note

The provided original CSV has 1,500 rows and 5 source columns. The raw project dataset includes `Payment Method`, giving the project 6 source attributes before preprocessing.

The raw project dataset includes realistic data quality issues for the cleaning demo:

- Blank descriptions, categories, and payment methods
- Inconsistent category, transaction type, and payment method formatting
- Currency-formatted amount strings
- Invalid amount values
- Invalid dates
- Duplicate rows

These issues are generated by `scripts/create_raw_dataset.py` so the dataset can be recreated consistently.
